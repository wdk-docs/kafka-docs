消息队列 Kafka¶

消息队列 Kafka（MQ for Kafka）秉持开放、共享的原则拥抱开源，全面融合 Kafka 开源生态，做到无缝迁移，打造更安全、更可靠、更易运维的 Kafka 企业级消息服务。
